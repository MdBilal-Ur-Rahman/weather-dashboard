# 🌦️ Weather Dashboard

A modern Full Stack Weather Dashboard built using **React, Vite, Node.js, Express, MongoDB, and OpenWeather API**.

It provides real-time weather information, hourly forecast, 5-day forecast, air quality, interactive maps, favorites, and recent searches.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/MdBilal-Ur-Rahman/weather-dashboard.git
```

Move into project

```bash
cd weather-dashboard
```

---

# Install Frontend

```bash
cd client
npm install
```

Run Frontend

```bash
npm run dev
```

---

# Install Backend

```bash
cd server
npm install
```

Run Backend

```bash
npm start
```

---

# Environment Variables

## Client (.env)

```env
VITE_API_URL=http://localhost:5000/api
```

---

## Server (.env)

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

WEATHER_API_KEY=YOUR_OPENWEATHER_API_KEY
```
