# Weather Dashboard

A modern and polished weather application built with React, Vite, Tailwind CSS, Framer Motion, Leaflet, and the OpenWeather API.

## Project Overview

This project is a premium-looking weather dashboard designed for portfolio presentation and recruiter evaluation. It combines a strong UI experience with real API integration, interactive maps, search handling, and saved user preferences.

## Key Features

- Live weather data for cities and villages
- Smart search with friendly feedback for invalid locations
- Interactive map with smooth pan and zoom controls
- Favorites and recent searches stored with local storage
- Responsive, glassmorphism-inspired interface

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Leaflet + react-leaflet
- OpenWeather API

## Project Structure

```text
src/
  components/     # Reusable UI components
  hooks/          # Custom React hooks
  services/       # API integration logic
  utils/          # Local storage helpers
  assets/         # Images and icons
```
